# Aenigma Press

Landing page bilingue (EN/IT) del marchio editoriale Aenigma Press,
pubblicata su [aenigmapress.com](https://aenigmapress.com) tramite
GitHub Pages.

Presidia il marchio, presenta il catalogo (collana Kakuro su Amazon
KDP) e raccoglie iscritti alla newsletter tramite MailerLite, con un
PDF gratuito di 10 puzzle Kakuro inediti come incentivo.

## Stack

HTML e CSS statici, zero build, zero dipendenze. Hosting GitHub Pages,
dominio Cloudflare, newsletter MailerLite.

## Struttura

- `index.html`: home in inglese (root)
- `it/index.html`: home in italiano
- `privacy.html`, `it/privacy.html`: pagina privacy
- `404.html`: pagina di errore
- `assets/style.css`, `assets/img/`: stile e immagini
- `CNAME`: dominio custom

Dettagli architetturali, regole di progetto e stato dei lavori:
vedi `CLAUDE.md`.
